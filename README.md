
### Change mongodb credentials 

- Get inside the `backend/database` folder
- Add mongodb url in `db.js` file

### Start App
- Run `npm install` to install required packages 
- Open terminal run `npm start`

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it on the browser.


### Start Node Server

- Get inside the `backend` folder
- Run `npm install` to install required packages 
- Open terminal run `node index.js`

Open API URL on [http://localhost:8022/api](http://localhost:8022/api)
